# MongoDB Basics with Mongoose

This project demonstrates **basic MongoDB CRUD operations** using **Node.js** and **Mongoose**.  
It is created for **learning and practice purposes** to understand how MongoDB works with Mongoose.

---

##  Features
- MongoDB connection using Mongoose
- Environment variables using dotenv
- Create, Read, Update, Delete (CRUD) operations
- Query examples (find, findOne, findById, sort, limit, skip)
- Count documents
- Schema & Model usage

---

## Technologies Used
- Node.js
- MongoDB Atlas
- Mongoose
- dotenv
- JavaScript
